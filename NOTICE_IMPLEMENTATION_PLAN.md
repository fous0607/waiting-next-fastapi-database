# 공지사항 대상 유형 확장 구현 계획

## 요구사항
1. **공지 대상 유형 확장**
   - 전체 매장 (all)
   - 일부 매장 (selected)
   - 프랜차이즈 공지 (franchise) - 특정 프랜차이즈의 모든 매장
   - 프로그램 공지 (program) - 시스템 전체 공지 (설정 가능)

2. **권한 기반 표시**
   - 프랜차이즈 공지: 해당 프랜차이즈 권한이 있는 사용자만 볼 수 있음
   - 프로그램 공지: 설정에 따라 표시 여부 결정

3. **프로그램 공지 설정**
   - 프로그램 공지에 대한 별도 설정 기능

## 구현 완료
✅ 백엔드 모델 수정 (Notice 모델에 franchise_id 추가)
✅ 백엔드 API 수정 (공지 생성/조회 시 franchise_id 처리)

   - 공지 목록에 대상 유형 표시

2. 프랜차이즈별 공지 조회 API 추가
