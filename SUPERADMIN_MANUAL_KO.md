# 슈퍼관리자 (Superadmin) 시스템 사용 가이드

본 문서는 대기열 관리 시스템의 **슈퍼관리자(System Admin)** 전용 기능 및 사용법을 설명합니다.

---

## 1. 개요
슈퍼관리자는 시스템의 모든 프랜차이즈, 매장, 사용자, 그리고 데이터를 관리할 수 있는 최고 권한 관리자입니다.
- **계정**: `superadmin` / `system_admin` 권한 보유

---

## 2. 주요 기능 및 변경 사항

### 2.1. 매장 설정 자동 잠금 해제 (Auto-unlock)
슈퍼관리자 계정의 편의성을 위해, 매장별 설정 페이지 진입 시 **보안 잠금이 자동으로 해제**됩니다.
- **기존**: 설정 변경을 위해 매번 매장별 '관리자 비밀번호' 입력 필요.
- **변경**: 슈퍼관리자는 로그인 즉시 인증된 상태로 간주되어, 별도의 비밀번호 입력 없이 모든 설정을 즉시 수정할 수 있습니다.
- **화면 표시**: "설정 잠금 해제" 버튼이 나타나지 않으며, 모든 입력 필드가 활성화됩니다.

### 2.2. 매장 보안 비밀번호 관리
슈퍼관리자는 각 매장의 '설정 보안 비밀번호'를 조회하고 직접 변경할 수 있습니다.
비밀번호 분실 시 슈퍼관리자가 이를 재설정해줄 수 있습니다.

1. **[시스템 관리]** > **[매장 관리]** 탭으로 이동합니다.
2. 매장 목록에서 수정하려는 매장의 **[정보 수정]** 버튼(또는 매장명 클릭)을 누릅니다.
3. **매장 정보 수정** 팝업창이 열립니다.
    - **매장명**: 매장 이름 수정 가능
    - **보안 비밀번호**: 현재 설정된 비밀번호가 표시되며, 원하는 값으로 수정 가능 (기본값: `1234`)
4. **[저장]** 버튼을 누르면 변경 사항이 즉시 적용됩니다.

---

## 3. 일반 관리 기능

### 3.1. 프랜차이즈 및 매장 관리
- **프랜차이즈 생성/관리**: 새로운 프랜차이즈 브랜드를 등록하고 관리할 수 있습니다.
- **매장 생성**: 각 프랜차이즈 하위에 새로운 매장을 개설할 수 있습니다. 매장 코드는 자동으로 발급됩니다 (예: `S001`).
- **상태 변경**: 프랜차이즈 또는 매장을 **활성화/비활성화** 처리할 수 있습니다.

### 3.2. 데이터 초기화 (Danger Zone)
시스템 테스트 또는 운영상 필요한 경우, 특정 매장의 데이터를 **영구적으로 삭제(초기화)**할 수 있습니다.
**주의**: 이 작업은 되돌릴 수 없습니다.

- **회원 정보 초기화**: 해당 매장의 모든 회원 데이터 및 대기 연결 정보를 삭제합니다.
- **대기 정보 초기화**: 현재 대기 중인 목록을 모두 비웁니다.
- **이력 정보 초기화**: 과거의 출석 기록, 마감 기록 등 통계 데이터를 삭제합니다.

---

## 4. 접속 방법
1. `/login` 페이지에서 슈퍼관리자 계정으로 로그인합니다.
2. 로그인 후 상단의 **[시스템 관리]** 버튼을 클릭하거나 `/admin` 경로로 접속하여 관리자 대시보드에 접근합니다.
