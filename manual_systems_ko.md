# 시스템 업그레이드 매뉴얼

새롭게 추가된 3가지 시스템(변경 이력, 헬스 대시보드, 설정 스냅샷)의 사용법을 안내합니다.

## 1. 변경 이력 (Audit Logging)
매장 설정 변경 사항을 추적할 수 있는 기능입니다. 누가, 언제, 어떤 설정을 변경했는지 확인할 수 있습니다.

### 사용 방법:
1. **매장 설정** 페이지로 이동합니다.
2. 상단 탭에서 **[변경 이력]**을 클릭합니다.
3. 최근 변경 사항 목록이 표시됩니다.
   - **시간**: 변경이 발생한 시간
   - **사용자**: 변경을 수행한 관리자
   - **유형/내용**: 변경된 항목 요약
   - **IP**: 접속 IP 주소

---

## 2. 헬스 대시보드 (Health Dashboard)
매장의 키오스크나 PC가 켜져 있는지(온라인 상태인지)를 실시간으로 모니터링합니다.

### 확인 방법:
1. **슈퍼어드민(Superadmin)** 계정으로 로그인합니다.
2. **매장 관리** 대시보드를 확인합니다.
3. 각 매장 카드에 상태 표시등이 나타납니다.
   - 🟢 **온라인**: 최근 5분 이내에 활동 감지됨
   - 🟡 **확인 필요**: 5분~30분 동안 활동 없음
   - 🔴 **오프라인**: 30분 이상 활동 없음 (기기 꺼짐 또는 인터넷 연결 끊김)

*참고: 대기관리자 화면(manage.html)이 켜져 있어야 신호를 보냅니다.*

---

## 3. 설정 스냅샷 (Configuration Snapshots)
매장 설정을 백업하고, 필요할 때 언제든지 이전 상태로 되돌릴 수 있는 기능입니다.

### 백업 생성 방법:
1. **매장 설정** > **[백업 관리]** 탭으로 이동합니다.
2. **백업 설명** 입력란에 간단한 메모(예: "업데이트 전 백업")를 입력합니다.
3. **[백업 생성]** 버튼을 클릭합니다.

### 설정 복원 방법:
1. **[백업 관리]** 탭의 **백업 목록**에서 복원하고 싶은 시점을 찾습니다.
2. 우측의 **[🔄 복원]** 버튼을 클릭합니다.
3. 확인 메시지에서 "확인"을 누르면 해당 시점의 설정으로 즉시 롤백됩니다.

> ⚠️ **주의**: 복원 시 현재의 모든 설정이 백업 시점의 내용으로 덮어씌워집니다. 필요한 경우 복원 전에 현재 상태를 먼저 백업하세요.
